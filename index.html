SERENA

Sensado, monitoreo y estudio del entorno sonoro para
la evaluación del impacto antropogénico en Antártida

Emiliano Acevedo, Maria Noel Espinosa, Ilana Stolovas

Resumen. El proyecto de fin de carrera SERENA surge en colaboración con el proyecto "Antartic Soundscapes" (AS) de Facultad de Ciencias (UdelaR), el cual tiene como objetivo estudiar el impacto antropogénico en el ambiente sonoro de la Antártida a partir de grabaciones del paisaje sonoro. Para cumplir con este cometido, el equipo del proyecto AS envía grabadores en las campañas de verano antárticas, los cuales son dispuestos en distintos puntos de la Isla Rey Jorge para muestrear el paisaje sonoro. En el transcurso del proyecto SERENA se implementaron algoritmos de aprendizaje automático para la identificación automática de fuentes sonoras de interés a partir de las grabaciones recolectadas. Los datos empleados para entrenar los modelos así como un análisis cualitativo de los resultados obtenidos se encuentran a continuación.

Audios Sintéticos


Debido a que no se contaba con una base de datos reales etiquetados se optó por generar una base sintética para entrenar los modelos. La misma fue generada con Scaper, una biblioteca de Python que crea nuevos paisajes sonoros a partir de combinar y transformar un conjunto de archivos de audio existentes. Como audios fuente se utilizaron fondos reales de Antártida los cuales fueron mezclados con sonidos de las fuentes que se desean identificar extraídos de AudioSet y Freesound.
