SERENA

Sensado, monitoreo y estudio del entorno sonoro para
la evaluación del impacto antropogénico en Antártida

Resumen. El proyecto de fin de carrera SERENA surge en colaboración con el proyecto "Antartic Soundscapes" de Facultad de Ciencias (UdelaR), el cual tiene como objetivo estudiar el impacto antropogénico en el ambiente sonoro de la Antártida a partir de grabaciones del paisaje sonoro. Durante el proyecto SERENA se implementaron algoritmos de aprendizaje automático para la identificación de fuentes sonoras de interés.

Audios Sintéticos


Para la generación de la base sintética se utilizó Scaper, una biblioteca de Python que crea nuevos paisajes sonoros a partir de combinar y transformar un conjunto de archivos de audio existentes. A continuación se muestran algunos ejemplos de los audios sintéticos generados.
